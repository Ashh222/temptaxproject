Filing client returns

Filing client returns online

To file on behalf of clients, you will need to register with HMRC as an agent. If you are not yet registered with HMRC for online filing, click here.

The HMRC online registration will enable you to use all of the following services:

  • Corporation Tax returns
  • Self Assessment returns
  • VAT for Agents
  • VAT EU Refunds for Agents
  • PAYE/CIS for Agents
  • Stamp Taxes for Agents

Printing and filing paper returns

HMRC would prefer you to file online. However, if you are unable to do so or would prefer to file a paper return, you can print and file a return using this software.

To see the requirements, penalties and exceptions for filing by paper, you can view HMRC guidance on filing by paper.

GOV.UK Fetch Taxpayer Data

Wherever you see a box like this, the software can collect information about income, expenses, allowances and more and populate the tax return.

Importing joint Income

As well as allowing you to bring your data forward from last year, this software includes options for importing data from another individual return.

Joint Investments, payments etc

You may own investments or make payments jointly with someone else. To save entering the same details twice, import data already entered onto another person's return.

Importing other Income

Import from self-employed accounts files or partnership returns where data has already been completed.

Self Employed accounts

If you have your accounts in an SNF file (Standard Name Format) then you can import that data into a specific set of self-employed pages.

Income from a Partnership

If you received income from a partnership and have completed an SA800 return then you can import your income and losses directly from the partnership return.
